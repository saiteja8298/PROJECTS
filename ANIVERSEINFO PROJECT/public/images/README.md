# Images Directory Structure

## Main Images

Place main anime character/series images here:

- **solo-leveling.jpg** - Solo Leveling artwork
- **naruto.jpg** - Naruto artwork  
- **one-piece.jpg** - One Piece artwork
- **profile.jpg** - Profile picture for About Me page

## Subdirectories

### `/wallpapers/`
Contains all background wallpapers (see wallpapers/README.md for details)

### `/history/`
Contains history card images:
- **1.jpg** - History item 1
- **2.jpg** - History item 2
- **3.jpg** - History item 3
- etc.

### `/preview/`
Contains preview images for generated manhwa:
- **preview-1.jpg**
- **preview-2.jpg**
- **preview-3.jpg**

## Image Sources

You can download anime images from:
- Official anime websites
- Wallpaper websites (Wallpaper Engine, Wallhaven)
- Fan art communities (with proper attribution)
- Anime screenshot collections

## Recommendations

For best results:
- Use high-quality images (at least 1920x1080 for wallpapers)
- Optimize images for web (compress but maintain quality)
- Use consistent art styles where possible
- Ensure images match the dark anime theme
